Raspberry Pi Tips

Raspberry Pi Tips

The raspberry pi is a computer that you can leave on since it consumes such low amounts of power. Here I make notes on the ways I configure my raspberry pi to be more productive.

  1. Enable SSH When you start up a raspberry pi for the first time, you need to attach it to a monitor with a hdmi cable. You can have the raspberry pi do your bidding remotely using SSH. You can enable it by opening the raspi-config menu and selecting the 'Interface Options'. To access the menus, use the command:
sudo raspi-config

  1. Using Tmux Sometimes it is useful to have a session running in the raspberry pi to come back to. For example, if you wanted to start a python script and then come back to it hours later, tmux helps you so that you don't stop the script from exiting SSH.

Install Tmux by:

sudo apt install tmux

Then after SSH'ing into your pi, start a tmux session by:

tmux

When you are done and want to disconnect from the pi:

Ctrl+b, d

to disconnect from the session.

To reattach to the session again and see the results of your script:

tmux attach-session -t 0
